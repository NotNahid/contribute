Here is the complete process you just went through. You can use this checklist for any open-source project in the future.

Phase 1: One-Time Setup (Per Computer)
Do this once when you open a new terminal.

Configure Identity:

Bash

Phase 2: Starting a Project
Fork the Repo: Click the "Fork" button on GitHub (e.g., on google/jaxite).

Clone to Computer:

Bash

git clone https://github.com/NotNahid/jaxite.git
cd jaxite
Phase 3: The Daily Work Cycle
Make Changes: Edit your files (README.md, code, etc.).

Stage & Commit: Save the changes to your local history.

Bash

git add .
git commit -m "Description of what I did"
(Tip: This is where the Author name is stamped. Since you did Phase 1, it will be correct now!)

Push to GitHub: Upload your saved history.

Bash

git push origin main
(Note: origin is the default nickname for your fork).

Phase 4: Submission
Create Pull Request: Go to GitHub, click "Compare & pull request," fill out the form, and submit.
